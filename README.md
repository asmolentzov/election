# Election

#Getting Started:
Clone this repository.
Insure node is install by running:
node -v

Install Truffle:
npm install -g truffle

Install Ganache

Install Metamask (chrome extension)
Connect Metamask.

Run the server:
npm run dev

Remix - debugging tool
https://remix.ethereum.org
